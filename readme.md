# 🥊 ChuckListJoke

¡Bienvenido a **ChuckListJoke**! Una aplicación web interactiva que rinde homenaje al único e inigualable Chuck Norris. Este proyecto está diseñado para poner en práctica los fundamentos del desarrollo Frontend modernos, estructurando el código de forma limpia, escalable y profesional.

---

# 📁 Estructura del Proyecto


CHUCKLISTJOKE/

├── css/                     # Estilos de la aplicación
│   └── styles.css
│
├── img/                     # Recursos visuales e imágenes
│   ├── chuck-icon.png       # Avatar / Icono de la app
│   └── chuck-norris.jpg     # Imagen principal
│
├── js/                      # Lógica y dinamismo (JavaScript)
│   └── script.js
│
├── index.html               # Estructura e interfaz principal (HTML5)
└── readme.md                # Documentación del proyecto

# Tecnologías Utilizadas
HTML5: Estructuración semántica del contenido de la página.

CSS3: Diseño responsivo, moderno y estilizado para una experiencia visual óptima.

JavaScript (ES6+): Manipulación del DOM, control de eventos y lógica interactiva.
